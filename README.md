# 🎮 GameChatTranslator v1.0.0

**GameChatTranslator**는 게임 화면 속의 다양한 언어(러시아어, 일본어, 중국어 등)를 실시간으로 인식하여 한국어로 번역해주는 윈도우용 오픈소스 도구입니다.

> **📌 프로젝트 배경**: 이 프로젝트는 아이드림스카이(iDreamSky)의 **스트리노바(Strinova)**를 플레이하며 외국인 유저들과의
>                       원활한 소통을 돕기 위해 제작된 간단한 개인 프로젝트입니다. 

> **💡 개발 정보**: 전반적인 로직 설계 및 성능 최적화는 **Google Gemini Pro**와 협업하여 진행되었습니다.

## ✨ 주요 기능 (Features)

- **실시간 OCR 번역**: Windows 전용 OCR 엔진을 사용하여 게임 화면의 텍스트를 빠르게 추출합니다.
- **다국어 지원**: 영어, 일본어, 중국어, 러시아어 등 다양한 언어를 한국어로 번역합니다.
- **자동 번역 모드**: 일정한 간격(기본 5초)으로 화면을 자동 스캔하여 번역 결과를 업데이트합니다.
- **사용자 커스텀**: `config.ini` 파일을 통해 불투명도, 단축키, 번역 간격 등을 자유롭게 설정할 수 있습니다.
- **게임 최적화 배치**: **스트리노바** 등 FPS 게임의 채팅창 위치(좌측 하단)에 맞춰 자동으로 배치됩니다.

## 🛠️ 시작하기 전 필수 설정

프로그램이 외국어를 정상적으로 인식하려면 윈도우에 해당 언어팩이 설치되어 있어야 합니다.

1. 압축 파일 내의 **`LangInstall.bat`** 파일을 **마우스 우클릭 > 관리자 권한으로 실행**하세요.
2. 설치가 완료되면 반드시 **컴퓨터 재부팅**을 진행해야 언어 인식이 활성화됩니다.
3. 프로그램을 실행할 때는 **관리자 권한으로 실행**해야 단축키와 화면 캡처가 원활하게 작동합니다.

## ⌨️ 기본 단축키 (Hotkeys)

| 기능 | 단축키 | 설명 |
|:---|:---|:---|
| **이동/잠금** | `Ctrl + 7` | 테두리가 녹색일 때 드래그하여 위치를 옮길 수 있습니다. |
| **영역 설정** | `Ctrl + 8` | 번역할 채팅창 범위를 마우스 드래그로 지정합니다. |
| **수동 번역** | `Ctrl + 9` | 현재 지정된 영역을 1회 즉시 번역합니다. |
| **자동 번역** | `Ctrl + 0` | 자동 번역 모드를 켜거나 끕니다. (상단에 상태 표시) |

## ⚙️ 설정 파일 (config.ini)

프로그램 실행 시 생성되는 `config.ini`에서 다음 값을 조정할 수 있습니다.

- `Opacity`: 0(투명) ~ 100(불투명) 사이의 값으로 번역창 투명도 조절
- `AutoTranslateInterval`: 자동 번역이 수행되는 주기(초 단위)
- `Threshold`: 폰트 인식 선명도 임계값 (기본 130)

## ⚠️ 주의 사항 (Disclaimer)

- 직접 제작한 배포 파일 특성상 윈도우 디펜더에서 차단될 수 있습니다. **[추가 정보] > [실행]**을 눌러 진행해 주세요.
- 게임이 **'전체 화면'** 모드일 경우 번역창이 보이지 않을 수 있으므로, **'창 모드'** 또는 **'전체 창 모드(테두리 없는 창)'** 설정을 권장합니다.
