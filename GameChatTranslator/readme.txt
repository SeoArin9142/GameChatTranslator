==========================================
   게임 실시간 채팅 번역기 (Game Translator)
   버전: v.1.0.7-alpha
==========================================

본 프로그램은 게임 화면의 채팅 영역을 캡처하고 Windows OCR로 글자를 인식한 뒤,
Google 무료 번역 또는 Gemini AI 번역을 사용해 한국어 등 원하는 언어로 표시하는 도구입니다.

현재는 스트리노바(Strinova)의 채팅창과 "[캐릭터명]: 채팅내용" 형식에 맞춰 개발 중인
초기 알파 버전입니다. 시스템 메시지, 귓속말, 대기실 채팅처럼 형식이 다른 문장은 번역 대상에서
제외될 수 있습니다.

------------------------------------------
1. 구성 파일 설명
------------------------------------------
- GameChatTranslator.exe
  프로그램 실행 파일입니다. 실행 시 관리자 권한 요청(UAC)이 표시됩니다.

- LangInstall.bat
  일본어, 중국어, 러시아어 OCR 언어팩 설치를 돕는 스크립트입니다.
  반드시 관리자 권한으로 실행해야 하며, 설치 후 재부팅이 필요합니다.

- characters.txt
  OCR 결과에서 캐릭터명을 검증할 때 사용하는 캐릭터명 목록입니다.
  이 파일에 없는 이름은 오인식 방지를 위해 번역에서 제외될 수 있습니다.

- config.ini
  프로그램 실행 후 자동 생성되는 설정 파일입니다.
  환경설정창에서 저장한 언어, 단축키, 캡처 영역, Gemini 키 등이 기록됩니다.

- logs 폴더
  실행 중 상태 메시지, OCR 원문, 번역 결과, 사용 엔진 정보를 날짜별 로그로 저장합니다.
  [OCR PERF] 항목에는 캡처, 확대, 전처리, OCR, 점수화, 번역 API 단계별 처리 시간이 기록됩니다.

- Captures 폴더
  "디버그 캡처 이미지 저장" 옵션을 켠 경우에만 생성됩니다.
  OCR 문제 분석용 원본/전처리/크롭 이미지가 저장됩니다.

------------------------------------------
2. 필수 사전 설정
------------------------------------------
1) LangInstall.bat 파일을 마우스 우클릭 후 [관리자 권한으로 실행]하세요.
2) 영어(en-US), 일본어(ja-JP), 중국어 간체(zh-CN), 러시아어(ru-RU) 중 필요한 언어만 Y로 선택하세요.
   예를 들어 영어/일본어/중국어만 필요하면 러시아어 질문에는 N을 입력하면 됩니다.
3) 선택한 언어팩 설치가 진행됩니다.
4) 설치 완료 후 반드시 Windows를 재부팅하세요.
5) 재부팅 후 프로그램을 실행하면 Windows OCR 엔진이 설치된 언어를 자동으로 감지합니다.

언어팩이 없으면 해당 언어 OCR 엔진을 만들 수 없어 외국어 채팅 인식률이 떨어집니다.
한국어와 영어만 사용할 경우에도 재부팅은 권장합니다.

------------------------------------------
3. 기본 실행 흐름
------------------------------------------
1) GameChatTranslator.exe를 실행합니다.
2) UAC 관리자 권한 요청이 표시되면 승인합니다.
3) 환경설정창에서 게임 언어, 번역 언어, 투명도, OCR 설정, 단축키를 확인합니다.
4) [저장 및 게임 시작] 버튼을 누르면 번역 오버레이가 표시됩니다.
5) 게임 채팅창 위치가 기본값과 다르면 Ctrl+8로 캡처 영역을 다시 지정합니다.
6) Ctrl+9로 수동 번역을 테스트한 뒤, Ctrl+0으로 자동 번역 모드를 켭니다.

------------------------------------------
4. 기본 단축키
------------------------------------------
- Ctrl + 7
  번역창 이동/잠금 전환입니다.
  잠금 상태에서는 마우스 클릭이 게임으로 통과합니다.
  이동 상태에서는 녹색 테두리가 표시되고 번역창을 드래그할 수 있습니다.

- Ctrl + 8
  번역할 채팅 영역을 다시 지정합니다.
  반투명 선택창이 뜨면 게임 채팅창을 마우스로 드래그하세요.

- Ctrl + 9
  1회 수동 번역입니다.
  품질 확인용이므로 OCR 정확 모드로 동작합니다.

- Ctrl + 6
  최근 번역 결과를 클립보드에 복사합니다.
  번역 결과가 없으면 로그에 안내 메시지를 남깁니다.

- Ctrl + 0
  자동 번역 모드를 순환합니다.
  순서: 빠름 -> 자동 -> 정확 -> OFF

- Ctrl + -
  Google 무료 번역과 Gemini AI 번역 엔진을 전환합니다.
  Gemini API 키가 없으면 전환할 수 없습니다.

------------------------------------------
5. 자동 번역 OCR 모드
------------------------------------------
- 빠름
  기본 Color 전처리와 게임 언어 OCR만 사용합니다.
  "[캐릭터명]: 채팅내용" 형식과 characters.txt 캐릭터명이 확인될 때만 번역합니다.
  처리 속도가 가장 빠르지만 배경이 복잡하면 인식 실패가 늘 수 있습니다.

- 자동
  먼저 빠름 모드로 시도합니다.
  실패한 경우에만 ColorThick, Adaptive 전처리와 다국어 OCR 후보를 추가 비교합니다.
  일반 사용 시 권장되는 균형 모드입니다.

- 정확
  모든 전처리 후보와 OCR 언어 결과를 비교합니다.
  배경 이펙트가 심하거나 투명 채팅창 뒤 배경이 밝을 때 유리하지만 처리 시간이 가장 깁니다.

- OFF
  자동 번역 타이머를 중지합니다.

------------------------------------------
6. OCR 관련 설정
------------------------------------------
- OCR 인식 배율(ScaleFactor)
  캡처 이미지를 몇 배로 확대해 OCR에 넣을지 정합니다.
  1배는 빠르지만 작은 글자를 놓칠 수 있고, 3배는 기본 권장값입니다.

- OCR 이진화 기준(Threshold)
  흰색 글자와 노란 캐릭터명을 배경에서 분리할 때 사용하는 기준값입니다.
  기본값은 120이며, 투명 배경에서 오인식이 심하면 110~130 사이로 조정해 보세요.

- 디버그 캡처 이미지 저장(SaveDebugImages)
  기본값은 꺼짐입니다.
  켜면 Captures 폴더에 Origin, Pre_Color, Pre_ColorThick, Pre_Adaptive, Crop 이미지가 저장됩니다.
  OCR 문제를 분석할 때만 켜는 것을 권장합니다.

------------------------------------------
7. Gemini AI 번역 설정
------------------------------------------
기본 번역 엔진은 Google 무료 번역입니다.
Gemini API 키를 입력하면 OCR 오타와 게임 문맥을 고려한 AI 번역을 사용할 수 있습니다.

설정 방법:
1) https://aistudio.google.com/ 에 접속합니다.
2) Google 계정으로 로그인합니다.
3) [Get API key] 메뉴에서 API 키를 생성합니다.
4) 환경설정창 하단의 Gemini AI 번역 설정에 API 키를 입력합니다.
5) Gemini Model은 기본값 gemini-2.5-flash를 사용하거나 필요한 모델명으로 바꿉니다.
6) 저장 후 실행하면 Ctrl+-로 Google/Gemini 엔진을 전환할 수 있습니다.

주의:
- API 키는 config.ini의 [Settings] GeminiKey에 저장됩니다.
- 프리셋에는 Gemini API 키가 저장되지 않습니다.
- API 사용량과 요금 정책은 Google 계정 설정을 확인하세요.

------------------------------------------
8. 환경설정창 주요 항목
------------------------------------------
- 설정 프리셋
  현재 설정을 이름별로 저장하고 다시 불러올 수 있습니다.
  언어, OCR 옵션, 단축키, 캡처 영역, 창 크기 등이 저장됩니다.

- 업데이트 확인
  실행 시 자동 업데이트 확인 여부를 설정합니다.
  새 버전이 있으면 릴리즈 페이지로 이동할지 묻고, 이동 시 프로그램은 종료됩니다.

- 언어 설정
  게임 언어는 OCR 기준 언어입니다.
  번역 언어는 최종 결과로 표시할 언어입니다.

- 단축키 설정
  입력칸을 클릭한 뒤 원하는 키 조합을 누르면 자동으로 문자열이 입력됩니다.
  [단축키 초기화] 버튼을 누르면 입력칸만 기본 단축키로 돌아갑니다.
  이 상태는 [저장 및 게임 시작] 버튼을 눌러야 config.ini에 저장됩니다.

- 캡처 영역
  저장된 영역을 확인하거나 초기화할 수 있습니다.
  실제 새 영역 지정은 Ctrl+8에서 진행합니다.

------------------------------------------
9. config.ini 주요 키
------------------------------------------
- GameLanguage
  OCR 기준 게임 언어입니다. 예: ko, en-US, zh-Hans-CN, ja, ru

- TargetLanguage
  번역 결과 언어입니다.

- Opacity
  번역창 불투명도입니다. 10~100 범위로 사용하는 것을 권장합니다.

- AutoTranslateInterval
  자동 번역 주기입니다. 단위는 초입니다.

- Threshold
  OCR 이진화 기준값입니다. 기본값은 120입니다.

- ScaleFactor
  OCR 입력 이미지 확대 배율입니다. 기본값은 3입니다.

- SaveDebugImages
  디버그 이미지 저장 여부입니다. true 또는 false.

- CheckUpdatesOnStartup
  실행 시 업데이트 자동 확인 여부입니다. false면 환경설정창 버튼으로만 확인합니다.

- GeminiKey
  Gemini API 키입니다.

- GeminiModel
  Gemini 호출 모델명입니다. 기본값은 gemini-2.5-flash입니다.

- Key_*
  각 기능의 단축키 설정입니다.

- Presets / Preset.이름
  프리셋 기능이 자동으로 관리하는 섹션입니다.

------------------------------------------
10. 문제 해결
------------------------------------------
- OCR이 외국어를 전혀 인식하지 못하는 경우
  LangInstall.bat 실행 후 재부팅했는지 확인하세요.

- 번역창 클릭이 되지 않는 경우
  Ctrl+7을 눌러 이동 모드로 전환하세요.
  잠금 상태에서는 게임 조작을 방해하지 않기 위해 클릭이 통과됩니다.

- 캡처 위치가 어긋나는 경우
  Ctrl+8로 영역을 다시 지정하세요.
  고DPI/멀티모니터 환경에서는 저장된 CapturePixelX/Y/W/H 값이 실제 캡처에 사용됩니다.

- 자동 번역이 느린 경우
  Ctrl+0으로 빠름 모드를 사용해 보세요.
  인식률이 부족하면 자동 또는 정확 모드로 전환하세요.
  logs 폴더의 [OCR PERF] 항목에서 OCR 시간이 느린지, Translate 시간이 느린지 먼저 확인하면 원인을 좁히기 쉽습니다.

- 밝은 배경 때문에 글자 인식이 흔들리는 경우
  정확 모드를 사용하거나 Threshold 값을 조정하세요.
  필요하면 디버그 캡처 이미지 저장을 켜고 Captures 폴더 이미지를 확인하세요.

- 업데이트 확인이 귀찮은 경우
  시작 시 업데이트 팝업에서 "다시 묻지 않기"를 선택하거나,
  환경설정창에서 "실행 시 업데이트 자동 확인" 체크를 해제하세요.

------------------------------------------
11. 주의 사항
------------------------------------------
- 본 프로그램은 알파 버전이며 모든 게임/해상도/배율 조합을 보장하지 않습니다.
- 게임은 전체 화면보다 전체 창 모드 또는 창 모드를 권장합니다.
- 관리자 권한을 거부하면 전역 단축키와 화면 캡처 기능이 정상 동작하지 않을 수 있습니다.
- Captures 폴더에는 게임 화면 일부가 저장될 수 있으므로 공유 전 개인정보를 확인하세요.
