==========================================
   게임 실시간 채팅 번역기 (Game Translator)
==========================================

본 프로그램은 게임 화면의 채팅 영역을 실시간으로 OCR 인식하여 
한국어로 번역해주는 도구입니다.

------------------------------------------
1. 필수 사전 설정 (중요!)
------------------------------------------
러시아어, 일본어, 중국어 등이 정상적으로 번역되려면 윈도우에 
해당 언어팩이 설치되어 있어야 합니다.

방법: 
1) 같이 동봉된 'LangInstall.bat' 파일을 [마우스 우클릭] -> [관리자 권한으로 실행] 하세요.
2) 검은 창이 뜨며 설치가 진행됩니다. (이미 설치된 경우 금방 끝납니다.)
3) 설치 완료 후 반드시 '컴퓨터 재부팅'을 해야 언어 인식이 정상 작동합니다.

------------------------------------------
2. 제미나이(Gemini) AI 번역 설정 (강력 추천!)
------------------------------------------
기본적으로 구글 무료 번역기가 내장되어 있으나, 게임 은어나 OCR 오타를 
문맥에 맞게 훨씬 자연스럽게 번역하려면 무료 Gemini API 키 적용을 강력히 권장합니다.

[API 키 발급 및 적용 방법]
1) 구글 AI 스튜디오( https://aistudio.google.com/ )에 접속 후 구글 계정으로 로그인합니다.
2) 좌측 상단의 [Get API key] 메뉴를 누르고 [Create API key] 버튼을 클릭합니다.
3) 'AIzaSy...' 로 시작하는 긴 문자열(API 키)이 생성되면 복사합니다.
4) 프로그램을 한 번 실행했다가 끈 뒤, 폴더에 생성된 'config.ini' 파일을 엽니다.
5) 아래와 같이 복사한 키를 붙여넣고 저장합니다.
   [Settings]
   GeminiKey=여기에_복사한_키를_붙여넣으세요
   GeminiModel=gemini-2.5-flash
6) 프로그램을 다시 실행하면 "적용 엔진: Gemini AI 번역" 이라는 로그와 함께 
   훨씬 똑똑해진 번역기가 작동합니다!

------------------------------------------
3. 실행 방법
------------------------------------------
- 'GameTranslator.exe'를 실행합니다.
- 단축키 등록 및 화면 캡처를 위해 [관리자 권한으로 실행]을 권장합니다.
- 처음 실행 시 좌측 하단에 자동으로 배치되며, 'config.ini' 설정 파일이 생성됩니다.

------------------------------------------
4. 기본 단축키 (변경 가능)
------------------------------------------
- [Ctrl + 7] : 번역창 이동/잠금 (녹색 테두리일 때 드래그 이동 가능)
- [Ctrl + 8] : 번역 영역 선택 (채팅창 범위를 마우스로 드래그하세요)
- [Ctrl + 9] : 1회 수동 번역
- [Ctrl + 0] : 자동 번역 시작/중지 (설정된 간격마다 실행)

------------------------------------------
5. 설정 변경 (config.ini)
------------------------------------------
프로그램 실행 폴더의 config.ini 파일을 열어 다음을 수정할 수 있습니다.
- Opacity: 번역창의 불투명도 조절 (0~100)
- AutoTranslateInterval: 자동 번역 간격(초)
- Key_... : 사용자 편의에 맞는 단축키 변경
- SaveDebugImages: 디버그 캡처 이미지 저장 여부 (true/false, 기본 false)

------------------------------------------
6. 주의 사항
------------------------------------------
- 윈도우 디펜더에서 '인식할 수 없는 앱'으로 차단될 경우, 
  [추가 정보] -> [실행]을 눌러주세요. 직접 제작한 프로그램이라 발생하는 현상입니다.
- 게임이 '전체 화면' 모드일 경우 번역창이 가려질 수 있으니 
  '전체 창 모드' 또는 '창 모드'로 플레이하는 것을 권장합니다.
